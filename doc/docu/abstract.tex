\begin{abstract}

\noindent Zur Analyse der immer größer werden Datenmengen, wird die Methode des Text-Minings immer wichtiger. 
Im vorliegenden Forschungsseminar wurde zu diesem Zweck die Verwendbarkeit von Hearst-Patterns zum Aufbau von Hyperonym-/Hyponymbeziehungen untersucht. 
Als erster Schritt wurde dazu ein geeignetes Tool gesucht auf Grund von Funktionsumfang und Performanz wurde sich für NLTK entschieden. 
Nachfolgend wurden ausgewählte Hearst-Patterns in NLTK implementiert und Testläufe mit diesen durchgeführt. 
Als Ergebnis dieser ersten Tests lässt sich festhalten, dass der ausschließliche Einsatz von Hearst-Patterns, Ergebnisse mit einer großen Anzahl falscher Beziehungen liefert 
und somit als alleiniges Mittel ungeeignet ist.
In einem anschließenden Forschungssemester sollen Methoden ermittelt werden diese Ergebnisse zu verfeinern.

\end{abstract}
